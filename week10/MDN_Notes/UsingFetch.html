MDN: Using Fetch

  • Fetch is a new way of getting and sending information accross the network.
  • At its basic from fetch is used like this: fetch(link to were you are getting or sending infromation).then(What you are going to do with the respose)
  • This is a good link for more information on supplying more options with your request.
  • You can upload JSON data, a file or even multiple files (using FormData()) using fetch.
  • Fetch only rejects the promise if a network error occured so you need to check that the respose was ok before handling the promise
  • You can create a request header that you than pass through fetch.
  • You can pass another request object into a request object and it will create a copy of it.
  • The Headers interface allows you to create your own headers object via the Headers() constructor. With the headers object you can query the header.
  • Response instances are returned when fetch() promises are resolved. The response contains response.status, response.statusText, response.ok and a body data which can contain differnt data

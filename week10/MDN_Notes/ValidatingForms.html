Week 10 Notes

MDN: Validating Forms

  • You can client-side validate forms using HTML5 or Javascript
  • For HTML 5 there is:
    1. Required attribute
    2. Regular Expressions
    3. Limiting the length of entries
    4. Setting the type attribute
  • For Javascript there is:
    1. Constraint Validation API
    2. Write the javascript from scratch
    3. Important notes always think of the user when doing validation. Make sure when there is an error the user knows where it is and what to do to fix it.

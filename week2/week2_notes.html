Data Types

Variables

  • const & let are used to assign variables
  • ex: const animal = mouse;
  • const variables can not be reassigned later
  • let variables can be reassigned later
  • var is not used but kept for backwards compatability
  • Constant and variable names can start with any upper or lower-case letter, an underscore, _ , or dollar character, $ . They can also contain numbers, but cannot start with them.
  • Names are case sensitive
  • Camel case is most used but underscore also works, just be consistant

Scope

  • Global scope is a variable that can be axcessed anywhere in the program
  • Local scope is a variable only axcessed in a block
  • To define gobal scope inside a block assign a value without using let or const

String Data Type

  • Use " or ' to surround, or alternate when wanting to display " or '
  • Has a wrapper class some of those methods are: indexOf(), charAt(), toLowerCase(), toUpperCase(), length, trim(), repeat(), includes().
  • A shortcut for string concatenation is to use the + operator to add the two strings together
  • Template Literals use ` for a string and can be used to put javacode right in the middle surround by ${}
  • The backslash is used to escape special characters in strings such as:
    Single quote marks \'
    Double quote marks \"
    End of line \n
    Carriage return \r
    Tab \t
  • To convert number to string use toString()

Symbols

  • Used for things like ids
  • Do not have a literal form

Numbers

  • If a number starts with a 0x , it is considered to be in hexadecimal (base 16) notation. Hexadecimal or "hex" numbers are often used for color codes on the Web
  • Number have methods as well
  • You can perform all sorts of arithmic with numbers
  • Heighest number:1.7976931348623157e+308 lowest is negative of that and will be refered to as infinity
  • NaN - not a number
  • You can use opperands on strings with numbers but is not recomened and as the out come isn't always expected
  • Use Number() to convert
  • parseInt() can also be used to retrive numbers from a string if you need an integer use parseFloat()

Array

  • Summary
  • Destructuringan array is the concept of taking values out of an array and presenting them as individual values. Ex: const [x,y] = [1,2];
  • Spread operator

Sets

  • Summary
  • Set is like an array but ignores multiples of the same value. After it is created it can be converted into an arry.
  • Weak sets - a way of storing data until the referance object is deleted

Maps

  • Summary
  • Maps are a collection of key value pairs and can be converted into an arry.
  • Weak maps - same concept as weak sets only for maps and limited on the methods
